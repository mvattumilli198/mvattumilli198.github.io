Executive Summary

The Horizon Retention & Success Hub is a modern early-alert analytics platform designed to help advisors identify at-risk students using 11 integrated datasets — including GPA trends, attendance, LMS activity, financial aid indicators, and engagement behaviors.

Using an XGBoost multi-class classifier (~92% accuracy), the system predicts risk tiers and displays them using a secure Spring Boot backend and a responsive React dashboard for advisors and students.

Problem Context & Stakeholders

Advisors

Need early alerts based on engagement + academic trends.

Students

Have fragmented data across LMS, grades, logs, and finances.

Leadership

Require visibility into patterns, performance, and risk clusters.

Proposed Solution

A unified analytics platform with dashboards, risk predictions, data insights, and structured advising support.
